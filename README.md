# how to run this project

## requirements to run the project:
* Docker or Podman
* Dotnet SDK 8.0
* Nodejs 24+
* NPM
    * @angular/cli


 "dotnet workload restore " or "sudo dotnet workload restore"
 "cd src/Frontends/chatbotweb" and "npm i"
 "dotnet run --project src/AppHost/AppHost.csproj"
